# Идея продукта: Авто рассылка СМС

## Проблема

Малый бизнес нуждается в регулярной (ежедневной) рассылке SMS-сообщений по своей клиентской базе. Существующие решения — платные сервисы и CRM-системы — требуют абонентской платы, передачи клиентской базы третьим лицам и дополнительной инфраструктуры. Простого, дешёвого и приватного инструмента для самостоятельной рассылки нет.

**Целевая аудитория:** владельцы малого бизнеса, которые ведут клиентскую базу в Excel и хотят рассылать персонализированные SMS своими силами.

## Решение

Десктопное Windows-приложение, которое работает в связке с Android-телефоном пользователя через USB (ADB). Пользователь загружает базу клиентов из Excel, составляет шаблон сообщения и запускает рассылку напрямую через свой телефон — без регистрации во внешних сервисах.

### Основная механика:
1. Телефон на Android подключается к ПК по USB с включённым режимом разработчика (ADB)
2. В приложении загружается Excel-таблица: столбец с номерами + столбец с переменным текстом
3. Конструктор собирает итоговый текст: **неизменяемая основа + переменная часть из таблицы**
4. Рассылка исполняется группами с настраиваемыми задержками для обхода спам-фильтров оператора
5. Приложение считывает статус доставки каждого сообщения из базы SMS-приложения телефона

## MVP (Minimum Viable Product)

1. **Загрузка базы из Excel** — импорт таблицы с двумя столбцами: номер телефона и переменный текст; автоматическая валидация формата номеров
2. **Конструктор сообщения** — редактор с полем для основного текста, вставкой переменной из таблицы и счётчиком символов; предпросмотр итогового сообщения
3. **Настройка отправки** — выбор размера группы (напр. 10 номеров), задержки между сообщениями (напр. 5 сек) и задержки между группами (напр. 1 мин)
4. **Запуск рассылки** — пошаговая отправка через ADB с отображением прогресса и статуса каждого сообщения (Отправлено / Доставлено / Ошибка)

## Уникальность

- **Нет абонентской платы** — используется SIM-карта самого пользователя
- **Полная приватность базы** — клиентские данные не покидают компьютер и телефон пользователя
- **Нет зависимости от внешних сервисов** — работает автономно, пока есть USB-кабель и мобильная связь
- **Простота** — не требует технических знаний: загрузил Excel, настроил, нажал «Старт»
